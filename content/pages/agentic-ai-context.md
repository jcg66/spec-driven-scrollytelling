# Context

This supporting page gives readers a short glossary for the homepage story without competing with it.

Use it before or after the main experience when the action-layer vocabulary needs a quick refresh.

## Vocabulary

- Intent: the user's goal or request.
- Planning: the way the agent turns intent into an ordered workflow.
- Grounding: the information the agent uses to stay aligned with what is visible.
- Execution: the actions the agent takes inside software.
- Outcome: the result the viewer should understand.

The action layer is the bridge between a request and a result.
