# Inside the Agentic Brain

## 1. Overall Topic
**Inside the Agentic Brain** is a visual deep-dive into the cognitive architecture of a 2026-era AI Agent. It explores how a model transitions from "thinking" (Large Language Model) to "acting" (Large Action Model). The narrative focuses on the three-step loop: **Perception** (seeing the screen), **Reasoning** (planning the steps), and **Action** (interfacing with the OS).

### Key Themes:
* **The Planner-Grounder Split:** How agents separate high-level goals from low-level clicks.
* **Visual Grounding:** The technology allowing AI to "see" a button without needing the underlying code.
* **MCP (Model Context Protocol):** The "nervous system" that allows agents to pull data from any app.

---

## 2. Story Outline
The website will be divided into four "chapters," triggered by the user's scroll depth.

* **Chapter 1: The Request (The Spark):** A user types a complex command: *"Find the cheapest flight to Tokyo and add it to my calendar if it's under $900."* We see the text convert into a "Goal Vector."
* **Chapter 2: The Planner (The Strategy):** The "Brain" splits the task. It realizes it needs to open a browser, navigate to a site, compare prices, and then access a second app (Calendar).
* **Chapter 3: Visual Grounding (The Vision):** The screen "zooms in" on the AI’s eye. We see a browser window through the AI’s perspective: buttons are highlighted with bounding boxes and "semantic labels."
* **Chapter 4: The Loop (The Execution):** A rapid-fire animation of the Agentic Loop—Action, Observation, Correction—until the task is marked "Success."

## 3. User Flow (Scrollytelling Mechanics)
| Scroll Depth | Visual Element | Interaction Type |
| :--- | :--- | :--- |
| **0% - 20%** | A floating, glowing "Core" representing the LLM. | **Parallax:** Text bubbles fly into the core as the user scrolls. |
| **20% - 50%** | The Core "unpacks" into a multi-layered map (The Planner). | **Sticky Narrator:** The map stays fixed while text explains "Chain-of-Thought" reasoning. |
| **50% - 80%** | A virtual desktop appears. An overlay shows the AI's "Visual Field." | **Cinematic Zoom:** The user scrolls "into" the browser window to see the AI's mouse movements. |
| **80% - 100%** | The "Goal Completed" checkmark appears. The UI dissolves into data points. | **Component Build:** The individual tools used (Browser, Calendar, MCP) reassemble into a "Success" badge. |
